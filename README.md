# Belaléa Web

React homepage for Belaléa with a design-matched interactive header.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Header features

- Belaléa logo (SVG)
- Navigation: О нас, КАТАЛОГ (dropdown), Диллерам, Новости
- Pill-shaped search bar with green search button
- Контакты contact button
- Hover and active states on all interactive elements

## Build

```bash
npm run build
```
